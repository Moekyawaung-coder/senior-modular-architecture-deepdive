# Senior Modular Architecture Deep Dive

This repository documents how a Senior Android Engineer designs, implements, and maintains large-scale modular applications.

**Key Topics Covered:**
- Feature modules vs Core modules vs Library modules
- Gradle configuration for 15+ modules
- Hilt multi-module setup with @InstallIn
- Baseline Profile per module
- R8/ProGuard rules per feature
- Module dependency graph visualization

**Live Example:** Linked to senior-android-multi-module repository.
